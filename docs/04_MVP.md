# MVP Scope

## Included
- Telegram authentication
- onboarding and preference settings
- buyer/seller profile
- listing drafts and publication
- multiple product images
- moderation
- personalized swipe feed
- favorites
- seller storefront
- conversations and text messages
- reserve and sold statuses
- reports, blocking and admin queue
- Vercel, Render and Supabase deployment

## Excluded from first release
- platform payments
- shipping integration
- returns and disputes
- seller subscriptions
- promoted listings
- advanced reviews
- machine-learning ranking
- native iOS/Android applications

## Definition of done
A new user can enter through Telegram, receive relevant active products, save one, contact the seller, and the seller can reserve or sell that product. All actions persist in PostgreSQL and can be moderated.
